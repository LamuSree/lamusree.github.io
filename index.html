Project Overview

This project demonstrates how to host a static website using Amazon S3. Amazon S3 provides scalable, secure, and highly available storage which can also be used to host static web applications.

AWS Services Used

  • Amazon S3 – Used to store and host static website files.
  • Amazon Route 53 – Used to manage DNS and custom domain names.
  • AWS Cloud – Provides scalable and reliable cloud infrastructure.

Project Architecture

User accesses the website through a browser. The domain name is managed by Route 53 which directs the request to the Amazon S3 bucket hosting the static website files.

User Browser → Route53 → Amazon S3 → Static Website

Conclusion

Amazon S3 static website hosting provides a simple and cost-effective way to deploy websites in the cloud. It ensures high availability, scalability, and easy management.
